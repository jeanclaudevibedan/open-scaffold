# Open Scaffold Release / Evidence Notes

This directory holds public release/evidence notes for meaningful Open Scaffold product slices.

A note should be short but reconstructable without chat history:

- roadmap item or milestone;
- GitHub issue or task ID;
- plan/spec path;
- run ID and run packet path, if generated;
- PR and merge/review status;
- verification commands and results;
- evidence paths and follow-up work.

GitHub Releases may become the public distribution surface later. These repo-local notes are the portable scaffold-native source of truth.
