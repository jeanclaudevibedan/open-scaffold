# Body-of-Work Wiki

## Definition

A body-of-work wiki is a scoped knowledge graph for durable project knowledge. It captures the concepts and reasoning future contributors and agents should not have to rediscover.

## Open Scaffold Use

For Open Scaffold, `docs/wiki/` is for public-safe compiled knowledge about the methodology and product thesis. It complements product docs and `.osc` execution files without replacing them.

## Does Belong Here

- project ontology;
- durable design principles;
- conceptual comparisons;
- reusable query answers;
- public-safe synthesis from accepted decisions.

## Does Not Belong Here

- active task state;
- PR status;
- raw runtime logs;
- private owner context;
- evidence receipts that belong in release/evidence surfaces.
