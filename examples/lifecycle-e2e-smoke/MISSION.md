# Mission

Tiny Notes is a tiny shell CLI that appends notes to a local text file so the Open Scaffold lifecycle can be tested without external services.

## Goals

- Add a note from the command line.
- Keep notes in a plain `notes.txt` file.
- Verify behavior with a local shell test.

## Non-Goals

Explicit things this project is NOT trying to do. Legitimate scope discipline starts here.

- Not a web app.
- Not a database-backed app.
- Not a hosted service.
